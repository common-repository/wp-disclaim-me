== Description ==

<p>WP-Disclaim-Me is a plugin that lets you create a disclaimer that a visitor to your blog / website must accept before they can access your posts and pages.</p>

<p>WP-Disclaim-Me has been tested in Wordpress version 2.5 or above. It is in beta so comments are very welcome!</p>

<strong>Features</strong>
<ul>
<li>Tested with Wordpress 2.8</li>
<li>Set your own disclaimer text from the admin pages</li>
<li>Set your own accept button text from the admin pages</li>
<li>NEW: WYSIWIG disclaimer text edit in the admin pages</li>
</ul>

<strong>COMING SOON</strong>

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `wp-disclaim-me` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set Your Disclaimer text and accept button text Admin > Settings > WP-Disclaim-Me
